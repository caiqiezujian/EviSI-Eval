# EviSI-Eval Agent Runbooks

Choose the language version:

- [English runbook](AGENT_RUNBOOK_EN.md)
- [中文运行手册](AGENT_RUNBOOK_ZH.md)

The English and Chinese versions describe the same v0.1 workflow:

1. prepare input JSONL files
2. build Evaluation Cards
3. run fact-level evaluation
4. apply deductions and cap rules
5. export HTML reports
6. add new labeled test cases
